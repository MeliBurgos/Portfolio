Sobre mí


sobre mí

Soy de Buenos Aires, Argentina. Comencé con un curso Introductorio de programación de Plataforma 5. Luego, me animé a realizar el Coding Bootcamp, donde aprendí a utilizar las herramientas más usadas. Se llevó a cabo varios proyectos, tanto de manera individual como en equipo, pudiendo así desarrollar las habilidades blandas. Lo cual fue un progreso en todos los aspectos. Dedicada, organizada, busco poder seguir avanzando y agregando experiencia. Me encuentro en la búsqueda de un puesto de Desarrollo Web y así poder aplicar mis conocimientos y crecer profesionalmente.


Herramientas

Proyecto freelance

Pagina web realizada desde cero para importante discoteca de Buenos Aires. Con vista para cliente: home, historia, cultura y vista administrador. Se aplicó metodologías ágiles: Scrum y las siguientes tecnologías: NodeJS, Express, Sequelize, PostgreSQL, React, Redux.

Black Cat

Tienda online de pastelería, en la cual el cliente puede agregar y quitar productos de su carrito, ver precio total y generar la compra. También tiene una vista de administrador. El proyecto se realizó en un período de dos semanas, durante el Coding Bootcamp, con un equipo de 5 personas. Se aplicó metodologías ágiles: Scrum y las siguientes tecnologías: NodeJS, Express, Sequelize, PostgreSQL, React, Redux.

Hornero App

Se desarrolló una aplicación web como práctica profesional durante el Coding Bootcamp. Se ubica y reserva oficinas o escritorios, con un equipo conformado por 5 personas. Donde se aplicó metodologías ágiles: Scrum y las siguientes tecnologías: NodeJS, MongoDB, Express, JWT, React, Redux.
